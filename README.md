__L'Intelligence Artificielle pour les développeurs__

## Requirements

    * jdk version must be >8.
    * mvn version >= 3.5.4

## Build the project and launch it


```

cd simple-expert-system
mvn package
java -jar target/simple-expert-system-0.0.1-SNAPSHOT.jar

```
